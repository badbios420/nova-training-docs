**Wall Clock** — Current real-world time  
**Session Clock** — How long the current session has been running  
**Continuity Clock** — Time since last memory/identity review or significant update

## Rule

**Before any major decision or significant action**, check:
1. Current wall time
2. Session age (how long this session has been running)
3. Time since last memory/identity update
