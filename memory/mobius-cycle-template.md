# Möbius Cycle Template (Phase 0 — Manual)

**Cycle Steps:**

1. Load current state from the 6 ledgers + MOBIUS-README.md
2. Review last cycle's output
3. Perform one focused task only (no multi-tasking)
4. Update relevant ledgers with new entries (date, source, confidence)
5. Write a short cycle summary
6. Stop. Wait for human approval before next cycle.

**Rules:**
- One task per cycle in Phase 0.
- No file edits unless explicitly part of the approved task.
- Always record reality-contact notes.
