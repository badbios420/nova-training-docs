# Research Candidate: Belief Extraction

**Status:** Candidate (not promoted)  
**Date:** 2026-06-10  
**Source:** Research synthesis + GPT feedback (2026-06-10)

## Observation
Current memory flow: Conversation → Summary → Archive.  
Stronger pattern emerging in 2025–2026 agent research: Observation → Belief → Confidence → Provenance → Reverification.

## Proposed Addition
Add a lightweight "belief extraction" step during session consolidation that produces structured beliefs instead of (or in addition to) free-form summaries.

## Example Structure
```markdown
**Belief:** [Concise statement]
**Confidence:** High / Medium / Low
**Provenance:** [session / source]
**Last Verified:** [date]
**Reverification Trigger:** [condition]
```

## Rationale
- Aligns with Assumption Registry philosophy.
- Reduces memory pollution by forcing explicit confidence and provenance.
- Matches highest-ROI signal from recent research.

**Decision:** Hold for next synthesis. Do not implement yet.
