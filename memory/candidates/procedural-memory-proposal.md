## Observation
Current durable memory contains:

- MEMORY.md (facts, decisions, discoveries)
- Assumption Registry

Missing category: **Procedural memory** — reusable methods for how Jason/Nova actually solve recurring problems.

## Examples of Missing Procedures
- How to recover after a provider crash
- How to perform a Reality Audit
- How to run a weekly synthesis

- How to execute the Daily Identity Check

## Proposed Structure
Create `memory/procedural-memory.md` (or equivalent) with entries in this format:

```markdown
## Procedure: [Name]

**Trigger:** When X happens...
**Steps:**
1. ...
2. ...

**Last Used:** [date]
**Effectiveness:** [notes]
```
